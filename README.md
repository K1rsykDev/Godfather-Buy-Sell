# Godfather Buy/Sell Web Form

Проста веб-сторінка для відправки оголошень «Продам» або «Куплю» до Discord каналу через Webhook.

## Використання
1. Відкрийте `script.js` та замініть значення `webhookUrl` на власний Discord Webhook.
2. Запустіть будь-який статичний сервер (наприклад, `npx serve` або через розширення Live Server) та відкрийте `index.html` у браузері.
3. На головній сторінці оберіть кнопку «Продам» або «Куплю», заповніть усі обов’язкові поля та додайте фото за потреби.
4. Після відправки дані надсилаються у Discord embed-повідомленням з вкладеним зображенням (якщо додано).

## Особливості
- Обов’язкові поля форми та проста валідація перед відправкою.
- Категорії товарів доступні для обох типів оголошень.
- Завантажене фото додається до повідомлення через вкладення Webhook.
- Адаптивний дизайн для мобільних пристроїв та ПК.
